You already know how to create empty and non-empty lists, so, naturally, at some point, you will want to manipulate your lists in some way. There is a large set of possible operations with lists that are fully described in the documentation. In this topic, we will consider only some of them.

Adding and removing elements

A list is a dynamic collection which means elements can always be added and removed. Let's start our consideration of such operations with creating an empty list of dragons.

dragons = []  # we do not have dragons yet

The first thing that comes to mind is, of course, adding new elements to the list.

To add a new element to the end of an existing list, you need to invoke list.append(element) function. It takes only a single argument, so this way you can add only one element to the list at a time. 

dragons.append('Rudror')
dragons.append('Targiss')
dragons.append('Coporth')

Now you have exactly three dragons, moreover, they are ordered the way you added them:

print(dragons)  # ['Rudror', 'Targiss', 'Coporth']

There is also list.extend(another_list) operation which adds all the elements from another iterable to the end of the list.

numbers = [1, 2, 3, 4, 5]
numbers.extend([10, 20, 30])
print(numbers)  # [1, 2, 3, 4, 5, 10, 20, 30]

If you use list.append(element) instead of list.extend(another_list), it will add the entire list, not its elements:

numbers = [1, 2, 3, 4, 5]
numbers.append([10, 20, 30])
print(numbers)  # [1, 2, 3, 4, 5, [10, 20, 30]]

Please, do not confuse these operations.

The opposite of adding elements — deleting them — is done with the help of list.remove(element) operation.

dragons.remove('Targiss')
print(dragons)  # ['Rudror', 'Coporth']

Now you have only two dragons in the list.

If the element you want to delete occurs several times in the list, then the first instance of that element is removed.

numbers = [0, 1, 1, 2, 3, 5, 8, 13, 21]
numbers.remove(1)
print(numbers)  # [0, 1, 2, 3, 5, 8, 13, 21]

 

Inserting elements at a specified position

We've learnt how to add new elements to the end of a list. If we want to add a new element in the middle of a list, we should use the list.insert(position, element) operation. The first argument is the index of the element before which to insert, so list.insert(0, element) inserts an element to the beginning of the list, and list.insert(len(a), element) is completely equivalent to list.append(element).

Here is an example:

years = [2016, 2018, 2019]
years.insert(1, 2017)  # [2016, 2017, 2018, 2019]
years.insert(0, 2015)  # [2015, 2016, 2017, 2018, 2019]
years.insert(len(years), 2020)  # [2015, 2016, 2017, 2018, 2019, 2020]

Now, you can fill any empty list with something useful :)

Reversing and sorting lists

There are two commonly-used operations to modify the order of elements in a list.

Invoking list.reverse() just reverses the order of elements in the list.

numbers = [1, 2, 3, 4, 5]
numbers.reverse()
print(numbers)  # [5, 4, 3, 2, 1]

The reverse() function is self-explanatory. Just mind that it performs the operation in place, i.e. it returns nothing, so you can't assign the result of this function to another variable, although the initial list will still be changed:

initial_list = [1, 2, 3, 4, 5]
reversed_list = initial_list.reverse()
print(reversed_list)  # None
print(initial_list)  # [5, 4, 3, 2, 1]

Invoking list.sort() sorts the list in the ascending order according to the natural order of stored elements.

numbers = [3, 2, 5, 4, 1]
numbers.sort()
print(numbers)  # [1, 2, 3, 4, 5]

To sort a list in the descending order, you need to specify reverse argument as True.

numbers = [3, 2, 5, 4, 1]
numbers.sort(reverse=True)
print(numbers)  # [5, 4, 3, 2, 1]

This function can also sort strings according to their lexicographic order (as in a dictionary).

strings = ['aa', 'b', 'aaa', 'q', 'qq']
strings.sort()
print(strings)  # ['aa', 'aaa', 'b', 'q', 'qq']

If your list has uncomparable types (like strings and integers together), it will throw an error:

TypeError: '<' not supported between instances of 'str' and 'int'

This concludes our discussion of the basic operations with lists. Check out the documentation if you need more information.
