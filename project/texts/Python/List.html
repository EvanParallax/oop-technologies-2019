Often you'll need to group several elements together so that you can process them as a single object. For that, you will need to use different collections, and one of the most useful in Python is a list. Understanding it is one of the keys knowledge in Python.

Creating and printing lists

Consider a simple list that stores several names of dogs' breeds:

dog_breeds = ['corgi', 'labrador', 'poodle', 'jack russel']
print(dog_breeds)  # ['corgi', 'labrador', 'poodle', 'jack russel']

In the first line, we use square brackets to create a list with four elements and then assign it to the dog_breeds variable. In the second line, the list is printed through the variable's name. All elements are printed in the same order as they stored in the list because lists are ordered.

Here is another list that contains five integers:

numbers = [1, 2, 3, 4, 5]
print(numbers)  #  [1, 2, 3, 4, 5]

Another way to create a list is to invoke the list function. This way is used when you want to create a list out of an iterable object: the kind of object where you can get its elements one by one. A list itself is iterable, so are other collections in Python, so is a string. 

The following example demonstrates the difference between using list and [] when creating a list:

multi_elements_list = list('danger!')
print(multi_elements_list)  # ['d', 'a', 'n', 'g', 'e', 'r', '!']

single_element_list = ['danger!']
print(single_element_list)  # ['danger!']

So, the list function creates a list containing each element from the given iterable object such as a string. You can use square brackets or invoke the list functions depending on your task.

The square brackets and the list function can also be used to create empty lists which do not have elements at all.

empty_list_1 = list()
empty_list_2 = []

In the following topics, we will consider how to fill empty lists using some operations.

Features of lists

Lists are able to store duplicate values as many times as needed.

on_off_list = ['on', 'off', 'on', 'off', 'on']
print(on_off_list)  # ['on', 'off', 'on', 'off', 'on']

Another important thing about lists is that they can contain different types of elements at the same time. So there are no restrictions, no fixed list types, and you can add whatever data you want to group together, like in the following example:

different_objects = ['a', 1, 'b', 2]

A list can also contain other lists as its elements (nested lists):

letters = ['a', 'b', ['c', 'd'], 'e']
list_of_lists = [[1], [2], [3]]
print(letters)  # ['a', 'b', ['c', 'd'], 'e']

Length of a list

Sometimes you need to know how many elements are there in a list. There is a built-in function called len that can be applied to any iterable object, and it returns simply the length of that object. So, when it's applied to a list, it returns the number of elements in that list.

numbers = [1, 2, 3, 4, 5]
print(len(numbers))  # 5

empty_list = list()
print(len(empty_list))  # 0

single_element_list = ['danger!']
print(len(single_element_list))  # 1

nested_list = ['dragon', ['cat', 'dog'], 'wyvern']
print(len(nested_list))  # 3

As you can see, the len of the nested_list is 3, because there are 3 elements in it: a string, a list, and another string.

Recap

As a recap, we note that lists are:

  • ordered, i.e. each element has a fixed position in the list;
  • able to store duplicate values;
  • able to store different types of elements at once, including other lists;
  • mutable, i.e. it is possible to add new elements or to remove existing ones.
