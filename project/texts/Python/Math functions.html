We already learned how to perform basic arithmetics in Python. We covered addition, substraction, multiplication, division and several other built-in operations. But if we want to do more complex operations on numbers we can use built-in mathematical functions or functions from the math module.

math module provides useful mathematical functions and constants. This module is available on every platform in standard library.

Advanced arithmetics

There are built-in functions abs, round, pow, max and min:

  • abs(x) returns the absolute value of x (i.e. value without a regard to its sign);
  • round(x, ndigits) returns x rounded to ndigits number of decimal part digits;
  • pow(x, y) returns x raised to the power of y;
  • max(a, b, c, ...) returns the largest argument;
  • min(a, b, c, ...) returns the  smallest argument.
abs_integer = abs(-10)  # 10
abs_float = abs(-10.0)  # 10.0

round_integer = round(10.0)  # 10, returns integer when ndigits is omitted
round_float = round(10.2573, 2)  # 10.26

pow_integer = pow(2, 10)  # 1024
pow_float = pow(2.0, 10)  # 1024.0

largest = max(1, 2, 3, 4, 5)  # 5
smallest = min(1, 2, 3, 4, 5)  # 1

abs and pow functions have equivalents in math module. The key difference of math.fabs and math.pow they always return floats:

import math

fabs_integer = math.fabs(-10)  # 10.0
fabs_float = math.fabs(-10.0)  # 10.0

pow_integer = math.pow(2, 10)  # 1024.0
pow_float = math.pow(2.0, 10)  # 1024.0

Remember that to use definitons from math you should import the module first.

Besides round function, we can use floor and ceil from math module to obtain integer values from floats:

  • math.floor(a) returns the nearest integer less than or equal to a;
  • math.ceil(a) returns the nearest integer greater than or equal to a.

Geometry

The number \(\pi\) is often used in geometry and other mathematical fields. It is the ratio of the circumference of a circle to its diameter. It can be found in math module as pi.

The next example shows how to calculate the circumference of a circle:

import math

r = 7
c = math.pi * r  # 21.991...

There are also common trigonometric functions available in the math module:

  • math.cos(a) returns the cosine of a radians;
  • math.sin(a) returns the sine of a radians;
  • math.degrees(a) returns angle a converted from radians to degrees;
  • math.radians(a) returns angle a converted from degrees to radians.
import math

deg = 60.0
x = math.radians(deg)  # 1.047...

cos = math.cos(x)  # 0.500...
sin = math.sin(x)  # 0.866...

rads = math.degrees(x)  # 59.999...

As you can see, due to a limited precision of floats the value of rads is actually 59.99999999999999 instead of expected 60.0.

The volume of a cylinder

Let's assume we have a cylinder with the height h = 5 and the radius of the base r = 3.  The formula for the volume of a cylinder is \(V = \pi r^2 h\). This is how we can calculate the volume using Python:

import math

h = 5
r = 3

volume = math.pi * math.pow(r, 2) * h  # 141.3716...

print(round(volume, 1))  # 141.4

In the code above we used a round function to get a prettier value for the printing.

As you see, it is possible to round a number or find a maximum value in Python using just built-in functions. However, for more advanced tasks you can use functions from math.
