What is iteration?

It’s common knowledge that computers are widely used to make the things people consider boring and energy-consuming. For example, repeating identical tasks without any errors. In Python, the process of repetitive execution of the same block of code is called an iteration.

There are two types of iteration:

Definite iteration, where the number of repetitions is written in advance.

Indefinite iteration, where the code block executes until the condition written in advance is true.

After the first iteration, the program comes back to the beginning of the code’s body and repeats it, making the so-called loop. The most used is the for loop, named after the for operator that provides the code’s execution.

For loop syntax

Here is the scheme of the loop:

for variable in iterable:
    statement

where statement is a block of operations executed for each item in iterable, an object used in iteration (e.g. string or list). Variable takes the value of the next iterable after each iteration.

Now guess which output we’ll get if we execute the following piece of code:

oceans = ['Atlantic', 'Pacific', 'Indian', 'Southern', 'Arctic']
for i in oceans:
    print(i)

During each iteration the program will take the items from the list and execute the statements with them, so the output will be:

Atlantic
Pacific
Indian
Southern
Arctic

Even strings are iterable, so you can spell the word, for example:

for a in 'magic':
    print(a)

 With the result:

m
a
g
i
c

Input data processing 

You can also use the input() method that helps the user introduce his/her own variable and work with it. Thus you can get the same output as with the previous piece of code:

word = input()
for a in word:
    print(a)

Have a look, you can write the code with a practical purpose:

times = int(input('How many times should I say "Hello"?'))
for i in range(times):
    print('Hello!')

You can, therefore, ask the user to specify the number of iterations to be performed.

Range method

The range() method is used to specify the number of iterations. It returns a sequence of numbers from 0 by default and ends at a specified number. Attention! The last number won’t be in the output, like in the example:

for i in range(5):
    print(i)

What we'll get is: 

0
1
2
3
4

Of course, it’s possible to adapt the method to your needs. You can change the starting value if you’re not satisfied with 0, moreover, you can configure the increment(step) value by adding a third parameter:

for i in range(5,30,5):
    print(i)

According to the parameters included, we’ve asked to print the numbers from 5 to 30 with an increment value of 5. Be careful again, the finishing value is not included in the output:

5
10
15
20
25

Nested loop

In Python, it is easy to use one loop inside another one – a nested loop. The type of inner and outer loops doesn’t matter, the first to execute is the outer loop, then the inner one executes:

names = ['Rose', 'Daniel']
surnames = ['Miller']
for x in names:
    for y in surnames:
         print(x, y)

The output is shown below: 

Rose Miller
Daniel Miller

In this example,  we use the two for loops to create fictional people's names. Obviously, you can deal with iterable objects of different sizes without too much fuss.

To sum up

All in all, for loops are a very efficient way to automate the code’s body repeating. You can add variables and operations to make a nested loop. Moreover, you can control the number of iterations with the help of the range() method. Be careful with the syntax: even extra indent or lack of colon can cause a mistake!
