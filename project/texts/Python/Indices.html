Since lists are ordered collections, each element has an index that corresponds to its position in the list. So, elements can be accessed by that index at any time.

Indices of elements

To access an element of a list by its index, you need to use the square brackets. You add the brackets after your list, and inside them you write the index of the element you're accessing.

Important, that the indices start with 0, so the first element has the index of 0. The last element has the index of len(list) - 1.

Here is an example.

colors = ['red', 'green', 'blue']

first_elem = colors[0]   # 'red'
second_elem = colors[1]  # 'green'
third_elem = colors[2]   # 'blue'

When using indices, it's important to stay within the range of your list: you'll get an error (called IndexError) if you try to access an element with a non-existing index!

colors = ['red', 'green', 'blue']
print(colors[3])  # IndexError: list index out of range

After some practice, you will not encounter such errors.

Accessing elements of nested lists

If a list has a nested list, elements of this nested list can be also accessed by indices.

numbers = [1, [2, 3], 4]
nested_numbers = numbers[1]

print(nested_numbers)  # [2, 3]
print(nested_numbers[1])  # 3

In this example, we obtain the second element of numbers by the index 1. This element is a nested list [2, 3]. Then we print the second element of nested_numbers by the index 1. This element is the number 3.

It is also possible to access an element of a nested list without an additional variable using a sequence of square brackets.

lists = [0, [1, [2, 3]]]
print(lists[1][1][0])  # 2

Now that you've learned what lists are and how to create them, you can use them in situations when you need to store many elements in a particular order.

Negative indices

To easily access the elements at the end of the list you can use negative indices: the minus before the number sort of changes your perspective and you look at the list from the end. So, the last element of the list, in this case, has an index equal -1, and the first element of the list has an index -length of the list.

Look at the examples below:

colors = ['red', 'green', 'blue']

last_elem = colors[-1]    # 'blue'
second_elem = colors[-2]  # 'green'
first_elem = colors[-3]   # 'red'

If you'll write a non-existing index, you'll also get IndexError.
