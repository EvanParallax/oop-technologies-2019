Simple if statement

In your future programs, you may need some actions to be performed only under certain conditions. Lines of code describing these actions should be placed within the body of an if statement. The pattern is the same as in the English language: the keyword if is followed by a condition, and after that, all actions necessary are listed. The condition is always a Boolean expression, that is, its value equals either True or False. Here is one example of how the code with a conditional expression should look like:

biscuits = 17
if biscuits >= 5:
    print("It's time for tea!")

Note that the condition is accompanied by a colon and there's an indentation at the beginning of a new line. Usually, 4 spaces are used to separate blocks of code. An if statement is executed only if its condition holds (the Boolean value is True), otherwise, it's skipped.

Nested if statement

Sometimes if statements are inserted into each other, then they are called nested. The number of if statements incorporated conduces to code complexity. At all times, it's better not to increase the current complexity of a code. However, this doesn't mean that nested if statements should be avoided at whatever cost. Let's take a look at the code below:

rainbow = "red, orange, yellow, green, blue, indigo, violet"
warm_colors = "red, yellow, orange"
cool_colors = "blue, green, indigo, violet"
my_color = "orange"
if my_color in rainbow:
    print("Wow, your color is in the rainbow!")
    if my_color in warm_colors:
        print("Oh, by the way, it's a warm color.")
    if my_color in cool_colors:
        print("Oh, by the way, it's a cool color.")

The example above illustrates a nested if statement. If the variable my_color is a string with the name of a rainbow color, we shall enter the body of the first if statement to print the first message and check whether this color pertains to the warm or cool colors. Here's what we'll see in our case:

Wow, your color is in the rainbow!
Oh, by the way, it's a warm color.

When it comes to nested if statements, proper indentation is crucial, so do not forget that each statement headed by the if keyword demands its own indention.
