As you may know, a string is just a sequence of characters enclosed in double quotes. In Kotlin, strings are represented by the String type. It is one of the most used data types in practice. In this lesson, we will learn the basic operations with strings.

The length of a string

It is possible to get the number of characters in a string accessing the length property. This property has a value of the Int type.

val language = "Kotlin"
println(language.length) // 6

val empty = ""
println(empty.length) // 0

Accessing characters

Elements of a string are individual characters that can be accessed by the index. The first element of a string has the index 0.

val greeting = "Hello"

val first = greeting[0]  // 'H'
val second = greeting[1] // 'e'
val five = greeting[4]   // 'o'

The last element has the index that equals the length of the string - 1. For the string "Hello" the last element is 'o'. The index of it is 4 because the string's length is 5.

val last = greeting[greeting.length - 1] // 'o'
val prelast = greeting[greeting.length - 2] // 'l'

Kotlin provides several convenient ways to access the first and the last character of a string as well as the last index:

println(greeting.first())   // 'H'
println(greeting.last())    // 'o'
println(greeting.lastIndex) // 4

Use this approach to make your code more readable.

Immutability

Strings are immutable that means once created the string is always the same. It is impossible to modify an element of a string. So the following code does not work:

val str = "string"
str[3] = 'o' // an error here!

But the following code works well:

var str = "string"
str = "strong"

Actually, we do not modify the stored value in the str variable. Instead, we assign a new value to it. So, it is absolutely legal. This is one of the approaches to work with strings. If you need a modified string - just create a new one.

Сoncatenating strings

Two string can be concatenated using the + operator:

val str1 = "ab"
val str2 = "cde"
val result = str1 + str2 // "abcde"

Important, when we concatenate two strings a new string is created (because strings are immutable).

It is possible to concatenate multiple strings in the same expression:

val firstName = "John"
val lastName = "Smith"
val fullName = firstName + " " + lastName

Note, str1 + str2 is not the same as str2 + str1 because the concatenation is not a commutative operation (unlike the addition).

Appending values to strings

The + operator also works for appending values of different types to a string. The value will be automatically converted to a string and then concatenated to the target string.
val str = "abc" + 10 + true
println(str) // abc10true

val code = "123" + 456 + "789"
println(code) // 123456789

Important, an expression must start with a string. So, the following code does not work at all:

val str = 10 + "abc" // an error here!

Maybe you will be surprised with the following code:

val str = "abc" + 11 + 22
println(str) // abc1122

The reason is first it appends 11 to the string "abc" and then appends 22 to the string "abc11".

Comparing strings

To compare two strings, use == (equals) and != (not equals) operators. Both operators take two strings as their operands and return a value of the Boolean type (true or false).

val first = "first"
val second = "second"
    
println(first == first)  // true
println(first == second) // false
println(first != second) // true

Conclusion

We explain only a few basic operations with strings. In fact, there are much more possible operations. We will study them in the next lessons.
