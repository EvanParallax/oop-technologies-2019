Kotlin provides the for loop to iterate through ranges, arrays and other collections of elements which we will learn in a future topic.

The for loop has the following base syntax:

for (element in source) {
    // body of loop
}

The body of this loop consists of one or more statements which are executed for each element from the specified source. The loop stops after the last element has been processed.

Iterating through a range

The simplest example of using the for loop is to print each element of an integer range.

for (i in 1..4) {
    println(i)    
}

This loops prints each individual number from 1 to 4.

1
2
3
4

It is also possible to iterate through a range of characters:

for (ch in 'a'..'c') {
    println(ch)
}

This loop prints:

a
b
c

Next, we will give all examples with numbers because iterating through characters is similarly.

Iterating in the backward order

It is also possible to iterate a range in the backward order.

for (i in 4 downTo 1) {
    println(i)
}

This loop prints numbers in the backward order from 4 to 1.

4
3
2
1

Important, it is required to use in 4 downTo 1, not in 4..1 to iterate the range in reverse order.

Excluding the upper limit

If we would like to exclude the upper limit from a range, we can subtract one from it or write until instead of ..:

for (i in 1 until 4) {
    println(i)
}

This loop prints numbers from 1 to 3.

Specifying a step

If we do not specify a step, it is assumed the step equal to one (e.g. 1, 2, 3, ...). But if we want to change the step, we should specify it explicitly.

In the following example, we print only odd numbers from the range 1..7.

for (i in 1..7 step 2) {
    println(i)
}

This loop prints four numbers:

1
3
5
7

It also can be used when we iterating in the backward order.

for (i in 7 downTo 1 step 2) {
    println(i)
}

This loop prints:

7
5
3
1

An example: the factorial of a number

As a classic example, let's write a program that calculates the factorial of a given integer number. The factorial of n is a production of integer numbers from 1 to n inclusive. Assumed that factorial of 0 is 1. The factorial of 1 is 1 as well.

import java.util.*

fun main(args: Array<String>) {
    val scanner = Scanner(System.`in`)

    val n = scanner.nextInt()
    var result = 1 // starting value of the factorial

    for (i in 2..n) { // the production from 2 to n
        result *= i
    }

    println(result)
}

The program above reads an integer number from the standard input. We suppose that the starting value of the factorial is 1 and then sequentially multiples it by numbers from 2 to n. If the input number is 1, the result is 1. If the input number is 5, the result is 120.

An example: the multiplication table of even numbers

It is possible to nest one for loop in the body of another for loop because loops are just regular statements. Nested loops are often used to process multidimensional structures like tables (matrices), data cubes and so on.

As an example, the following code prints the multiplication table of even numbers from 1 to from 10.

fun main(args: Array<String>) {
    for (i in 2..10 step 2) {
        for (j in 2..10 step 2) {
            print("${i * j}\t") // print the production of i and j followed by one tab
        }
        println()
    }
}

It prints:

4   8   12  16  20  
8   16  24  32  40  
12  24  36  48  60  
16  32  48  64  80  
20  40  60  80  100
